# Studio Book Frontend (ASP.NET Core / Next.js)

個人学習・ポートフォリオ用に作成している、書籍・学習サービスを想定したフロントエンド試作です。  
ASP.NET Core版とNext.js版の画面構成・UI表現の比較や、画面実装の練習を目的としています。

## 目的
- フロントエンド画面の実装練習
- ASP.NET Core版とNext.js版の比較
- ログイン画面、トップページ、一覧画面などのUI検証
- ポートフォリオ用の画面サンプル作成

## 主な内容
- ログイン画面
- トップページ
- 共通レイアウト（ヘッダー / フッター）
- 読み込み中表示
- 画面モック・UI調整

## 技術スタック
- Next.js
- TypeScript
- React
- CSS

## 備考
- 本リポジトリは学習・検証用です
- 実運用は想定していません
- 実在するサービス・データとは関係ありません
