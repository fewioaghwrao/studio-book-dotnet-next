# Studio Book Backend

書籍・学習サービスを想定した、個人学習・ポートフォリオ用のバックエンド試作です。  
会員登録、ログイン、書籍一覧、レビュー、予約などの機能を段階的に実装することを想定しています。

## 目的
- バックエンドAPI実装の練習
- Entity / Service / Controller の構成整理
- 認証・認可を含む基本機能の検証
- フロントエンド連携を前提としたAPI設計の学習

## 主な内容
- 会員登録
- ログイン認証
- 書籍・店舗・予約関連API
- 管理者向け機能
- エラーハンドリング

## 技術スタック
- ASP.NET Core
- C#
- Entity Framework Core
- SQL Server

## 備考
- 本リポジトリは学習・検証用です
- 実運用は想定していません
- 実在するサービス・データとは関係ありません
